Biografi HTML

HTML (HyperText Markup Language) pertama kali ditemukan oleh Tim Berners-Lee pada tahun 1991. HTML merupakan bahasa markup yang digunakan untuk membuat halaman web. Penemuan ini menjadi dasar pengembangan World Wide Web (WWW) yang saat ini digunakan di seluruh dunia.

HTML memungkinkan pembuatan struktur teks, gambar, tautan, dan elemen-elemen lain di dalam halaman web. Setiap halaman web yang kita lihat hari ini, di berbagai browser, dibuat dengan HTML.
