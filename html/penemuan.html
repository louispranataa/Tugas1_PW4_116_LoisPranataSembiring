Penciptaan HTML

HTML diciptakan oleh Tim Berners-Lee sebagai bagian dari proyek World Wide Web di CERN. Tujuannya adalah untuk mempermudah peneliti berbagi dokumen dengan format standar yang dapat diakses melalui internet.

Standarisasi HTML

Sejak pertama kali diciptakan, HTML terus dikembangkan dan distandarisasi oleh World Wide Web Consortium (W3C) untuk memastikan kompatibilitas dan interoperabilitas halaman web di berbagai platform dan perangkat.
