HTML Versi 1.0

HTML 1.0 diperkenalkan pada tahun 1991 oleh Tim Berners-Lee. Pada versi ini, HTML hanya mendukung elemen-elemen dasar seperti heading, paragraf, dan tautan. Ini adalah versi awal yang digunakan untuk membuat halaman web pertama yang sangat sederhana.

HTML Versi 2.0

HTML 2.0 dirilis pada tahun 1995 oleh Internet Engineering Task Force (IETF). Versi ini menambahkan fitur-fitur penting seperti formulir (form), yang memungkinkan pengguna untuk berinteraksi dengan halaman web melalui input data.

HTML Versi 4.01

HTML 4.01 diluncurkan pada tahun 1999 dan menjadi salah satu versi yang paling banyak digunakan. Versi ini membawa banyak peningkatan termasuk dukungan untuk skrip, tabel, dan elemen multimedia, yang menjadikan web lebih interaktif dan menarik.

HTML5

HTML5, yang dirilis secara resmi pada tahun 2014, membawa perubahan besar dalam pengembangan web. Fitur seperti elemen video dan audio, canvas untuk grafis, serta penyimpanan offline menjadikan HTML5 lebih fleksibel dan mendukung aplikasi web modern.
